GemFireXD Extended SQL Functions

Created December 2014
Pas Apicella - Pivotal Senior System Engineer

GemFireXD SQL String functions extend the built in SQL functions with popular functions based on Oracle SQL Functions such as INITCAP, REPLACE, TRANSLATE, LPAD, RPAD which are not part of the built in functions provided out of the box with GemFireXD

Example:

sqlf> select initcap('PAS APICELLA') from sysibm.sysdummy1;
1                                                                                                                              
-----------------------------------------------------------------------------------------------------------
Pas Apicella                                                                                                                   

1 row selected

Install Instructions

1. Download the following files

commons-lang3-3.1.jar              
gemfirexd-sqlextended-functions.jar
gemfirexd-sqlextended-functions.sql
test.sql

2. Place the following 2 jars into CLASSPATH for server nodes from downloads in #1
  • commons-lang3-3.1.jar              
  • gemfirexd-sqlextended-functions.jar
3. Run the following SQL in your target schema from the downloads in #1
  • gemfirexd-sqlextended-functions.sql
  • test.sql
Overview of SQL Functions

INITCAP

INITCAP function sets the first character in each word to uppercase and the rest to lowercase.
Usage: INITCAP(input_string)
sqlf> select initcap('PAS APICELLA') from sysibm.sysdummy1;
1                                                                                                                              
-----------------------------------------------------------------------------------------------------------
Pas Apicella                                                                                                                   

1 row selected

REPLACE

REPLACE function replaces a sequence of characters in a string with another set of characters

Usage: REPLACE( input_string, string_to_replace, replacement_string)

sqlf> select replace('aabbccaa', 'aa', 'zz') from sysibm.sysdummy1;
1                                                                                                                              
-------------------------------------------------------------------------------------------------------------
zzbbcczz                                                                                                                       

1 row selected

REMOVE

REMOVE function removes a sequence of characters in a string

Usage: REMOVE (input_string, string_to_remove)

sqlf> select replace('aabbccaa', 'aa', 'zz') from sysibm.sysdummy1;
1                                                                                                                              
-------------------------------------------------------------------------------------------------------------
zzbbcczz                                                                                                                       

1 row selected

GFXD_TRANSLATE

Note: This function is called GPXF_TRANSLATE as TRANSLATE is a reserved word in Derby

GFXD_TRANSLATE function replaces a sequence of characters in a string with another set of characters. However, it replaces a single character at a time. For example, it will replace the 1st character in the string_to_replace with the 1st character in the replacement_string. Then it will replace the 2nd character in the string_to_replace with the 2nd character in the replacement_string, and so on.

Usage: GFXD_TRANSLATE( input_string, string_to_replace, replacement_string )

sqlf> select gfxd_translate('222tech', '2ec', '3it') from sysibm.sysdummy1;
1                                                                                                                              
-----------------------------------------------------------------------------------------------------------------
333tith 

1 row selected

LPAD

LPAD function pads the left-side of a string with a specific set of characters

Usage: LPAD( input_string, padded_length, pad_string)
sqlf> select lpad('pas', 5, '') from sysibm.sysdummy1;
1                                                                                                                              
------------------------------------------------------------------------------------------------------------------
  pas                                                                                                                          

1 row selected

RPAD

RPAD function pads the right-side of a string with a specific set of characters

Usage:  RPAD( input_string, padded_length, pad_string)

sqlf> select rpad('pas', 5, '.') from sysibm.sysdummy1;
1                                                                                                                              
-------------------------------------------------------------------------------------------------------------------
pas..                                                                                                                          

1 row selected

INSTR

INSTR function returns the location of a substring in a string

Usage:  INSTR( string, substring, start_position, nth_appearance)
sqlf> select instr('Tech on the net', 'e', 1, 2) from sysibm.sysdummy1;
1          
-----------
11         

1 row selected
